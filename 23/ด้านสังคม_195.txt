รัฐบาลไทย-ข่าวทำเนียบรัฐบาล-กระทรวงดิจิทัลฯ กระตุ้นเด็กไทยพัฒนาทักษะเขียนโค้ตขั้นพื้นฐานผ่าน Code Thailand

วันจันทร์ที่ 3 ธันวาคม 2561
กระทรวงดิจิทัลฯ กระตุ้นเด็กไทยพัฒนาทักษะเขียนโค้ตขั้นพื้นฐานผ่าน Code Thailand
กระทรวงดิจิทัลฯ
ดร.พิเชฐ ดุรงคเวโรจน์ รัฐมนตรีว่าการกระทรวงดิจิทัลเพื่อเศรษฐกิจและสังคม ร่วมให้กำลังใจเด็กๆ ที่เข้ากิจกรรม Hour of Code Thailand 2018 ณ อาคารซีอาร์ซี ออลซีซันส์เพลส ชั้น 38 กรุงเทพฯ เมื่อวันที่ 2 ธันวาคม 2561 จัดขึ้นโดยสำนักงานส่งเสริมเศรษฐกิจดิจิทัล (DEPA Thailand) ร่วมกับ บริษัท ไมโครซอฟท์ (ประเทศไทย) จำกัด หรือ Microsoft (Thailand) Limited เพื่อกระตุ้นเยาวชนไทยให้มีความสนใจด้านวิทยาศาสตร์คอมพิวเตอร์ผ่านการเขียนโค้ดขั้นพื้นฐาน  เพราะในยุคของเศรษฐกิจดิจิทัล วิทยาศาสตร์คอมพิวเตอร์จะช่วยสร้างและส่งเสริมทักษะด้านการวิเคราะห์ ซึ่งต้องใช้ในการประกอบอาชีพในทุกสาขา ไม่ว่าจะเป็นดนตรี แฟชั่น การผลิตสินค้า การแพทย์และการคมนาคมขนส่ง ไม่ใช่แต่เฉพาะอาชีพที่เกี่ยวกับคอมพิวเตอร์เท่านั้น เพราะทักษะการโค้ดดิ้ง กระตุ้นให้เกิดความคิดสร้างสรรค์ การฝึกใช้เหตุผลวิเคราะห์และการแก้ปัญหาที่ซับซ้อน ซึ่งเป็นทักษะที่สำคัญในการประกอบอาชีพต่างๆ ผู้ที่สนใจสามารถเรียนรู้ได้ที่ https://www.codingthailand.org
