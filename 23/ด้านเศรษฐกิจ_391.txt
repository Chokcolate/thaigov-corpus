รัฐบาลไทย-ข่าวทำเนียบรัฐบาล-กรุงไทยร่วมมือกับ NETS และ ITMX พัฒนาการให้บริการระบบชำระเงินข้ามพรมแดน

วันอังคารที่ 27 พฤศจิกายน 2561
กรุงไทยร่วมมือกับ NETS และ ITMX พัฒนาการให้บริการระบบชำระเงินข้ามพรมแดน
ธนาคารกรุงไทยเข้าร่วมโครงการความร่วมมือระหว่าง NETS และ ITMX  ที่มีวัตถุประสงค์เพื่อแลกเปลี่ยนความคิดและพัฒนาเทคโนโลยีการชำระเงินข้ามพรมแดนระหว่างสองประเทศ ผ่านระบบ QR Code มาตรฐานของ NETS ประเทศสิงคโปร์
ธนาคารกรุงไทยเข้าร่วมโครงการความร่วมมือระหว่าง NETS และ ITMX  ที่มีวัตถุประสงค์เพื่อแลกเปลี่ยนความคิดและพัฒนาเทคโนโลยีการชำระเงินข้ามพรมแดนระหว่างสองประเทศ ผ่านระบบ QR Code มาตรฐานของ NETS ประเทศสิงคโปร์  และ QR Code มาตรฐาน ITMX ของประเทศไทย ที่งาน Fintech Festival ประเทศสิงคโปร์ ซึ่งความร่วมมือในครั้งนี้ เป็นก้าวแรกในการวางโครงสร้างพื้นฐานที่สำคัญในการสนับสนุนระบบการโอนเงินและชำระเงินระหว่างประเทศ โดยธนาคารกรุงไทยได้เล็งเห็นความสำคัญและนำ Mobile Banking App “Krungthai NEXT” เข้าร่วมทดสอบการชำระเงินผ่านการสแกน QR Code ของประเทศสิงคโปร์ และการรับชำระเงินจาก Singapore Application ผ่าน  Application “เป๋าตุง”  สำหรับร้านค้า
เน็ตส์ กรุ๊ป (NETS)  กลุ่มบริษัทเกี่ยวกับธุรกรรมการชำระเงินชั้นนำ ผู้ให้บริการชำระเงินผ่านระบบดิจิทัลซึ่งครอบคลุมการชำระเงิน และเป็นผู้ดำเนินโครงการชำระเงินให้กับองค์กรชั้นนำของประเทศสิงคโปร์ และบริษัท เนชั่นแนล ไอทีเอ็มเอ๊กซ์ จำกัด (ITMX) เป็นผู้วางโครงสร้างพื้นฐานระบบการชำระเงิน การโอนเงินระหว่างธนาคาร และเป็นศูนย์กลางข้อมูลระบบการทำธุรกรรมของประเทศไทย การร่วมลงนามในบันทึกข้อตกลง เพื่อแลกเปลี่ยนและพัฒนาเทคโนโลยีการชำระเงินข้ามพรมแดน (Cross-Border Usage & Acceptance) ของระบบชำระเงิน ซึ่งทั้ง 2 องค์กร เป็นสมาชิกเครือข่ายการชำระเงินภูมิภาคอาเซียนในครั้งนี้ เป็นจุดเริ่มต้นในการเชื่อมโยงธุรกรรมระหว่างประเทศในกลุ่มอาเซียน เป็นไปด้วยความสะดวก รวดเร็ว และปลอดภัยมากยิ่งขึ้น
มร.เจ็ฟฟรีย์ โก ผู้บริหาร เน็ตส์ กรุ๊ป กล่าวว่า การร่วมมือในครั้งนี้เกิดจากความมุ่งมั่นในการขยายการให้บริการชำระเงินผ่านระบบดิจิทัล เพื่อความสะดวกสบายของลูกค้าและผู้ประกอบการ ซึ่งคาดหวังว่าจะมีการแลกเปลี่ยนองค์ความรู้และความเชี่ยวชาญที่มีประโยชน์ อันจะช่วยสร้างประสบการณ์การชำระเงินแบบไร้ข้อจำกัดให้แก่ลูกค้าและภาคธุรกิจของทั้งสองประเทศ และขับเคลื่อนนโยบาย Smart Nation ของรัฐบาล
นางสาววรรณา นพอาภรณ์ กรรมการผู้จัดการ บริษัทเนชั่นแนล ไอทีเอ็มเอ๊กซ์ จำกัด  เปิดเผยว่า ความร่วมมือกับ NETS ในการให้บริการชำระเงินระหว่างประเทศในครั้งนี้ จะช่วยขยายการเข้าถึงกลุ่มลูกค้าและอำนวยความสะดวกแก่ผู้ใช้บริการทั้ง 2 ประเทศ ตลอดจนวางโครงสร้างพื้นฐานที่สำคัญในการสนับสนุนระบบการโอนเงินระหว่างภูมิภาคของประเทศไทย ให้สามารถเชื่อมต่อกับภูมิภาคอาเซียนและในระดับโลก
นายผยง ศรีวณิช กรรมการผู้จัดการใหญ่ ธนาคารกรุงไทย ได้ตอกย้ำนโยบายการชำระเงินข้ามพรมแดนว่า ถือเป็นหนึ่งในยุทธศาสตร์ NEXT to Invisible Banking ที่ธนาคารให้ความสำคัญ ซึ่งจะทำให้ลูกค้าได้รับความสะดวก ความปลอดภัย และจากการที่ธนาคารมีบริการครอบคลุมมากที่สุดในประเทศไทย ส่งผลให้ความร่วมมือในครั้งนี้ ตอบโจทย์ความต้องการของลูกค้าทุกกลุ่ม ซึ่งจะร่วมกันพัฒนาให้ผู้ใช้บริการ Mobile Banking ของประเทศไทย สามารถชำระเงินผ่านการสแกน QR Code ของ NETS ซึ่งเป็นส่วนหนึ่งของ QR Code สิงคโปร์ (SGQR) และผู้ใช้งาน Mobile Banking ของสิงคโปร์ สามารถชำระเงินผ่านการสแกน QR Code มาตรฐานของประเทศไทย ในการชำระค่าสินค้าและบริการข้ามพรมแดนได้อย่างสะดวกรวดเร็ว
