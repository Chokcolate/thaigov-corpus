รัฐบาลไทย-ข่าวทำเนียบรัฐบาล-รัฐบาลจัดงาน “ประชารัฐร่วมใจ ใต้ร่มพระบารมี”ช่วยเหลือผู้ประสบภัยน้ำท่วม

วันศุกร์ที่ 4 สิงหาคม 2560
รัฐบาลจัดงาน “ประชารัฐร่วมใจ ใต้ร่มพระบารมี”ช่วยเหลือผู้ประสบภัยน้ำท่วม
รัฐบาล โดยสำนักนายกรัฐมนตรี จัดงาน “ประชารัฐร่วมใจ ใต้ร่มพระบารมี” ช่วยเหลือผู้ประสบภัยน้ำท่วม ประชาชนทุกภาคส่วนแสดงน้ำใจร่วมบริจาคเงินเข้า “กองทุนเงินช่วยเหลือผู้ประสบสาธารณภัย สำนักนายกรัฐมนตรี” อย่างต่อเนื่อง
วันนี้ (4 ส.ค.60) เวลา 18.00 น. ณ ตึกสันติไมตรี (หลังนอก) ทำเนียบรัฐบาล พลเอก ประยุทธ์ จันทร์ โอชา นายกรัฐมนตรี เป็นประธานพิธี “งานประชารัฐร่วมใจ ใต้ร่มพระบารมี” จัดโดยรัฐบาล สำนักนายกรัฐมนตรี เพื่อเปิดโอกาสให้ประชาชนทุกภาคส่วน ทั้งหน่วยงานภาครัฐ รัฐวิสาหกิจ ภาคธุรกิจเอกชน และประชาชนทั่วไป มีส่วนร่วมและแสดงน้ำใจในการช่วยเหลือผู้ประสบภัยน้ำท่วมในภาคเหนือตอนล่างและภาคตะวันออกเฉียงเหนือ โดยบริจาคเงินสบทบเข้า “กองทุนเงินช่วยเหลือผู้ประสบสาธารณภัย สำนักนายกรัฐมนตรี” ผ่านบัญชีธนาคารกรุงไทย จำกัด (มหาชน) สาขาทำเนียบรัฐบาล เลขที่บัญชี 067 - 0 - 06895 - 0  เพื่อรัฐบาลจะได้นำเงินบริจาคไปช่วยเหลือประชาชนที่ได้ผลกระทบและความเดือดร้อนจากสถานการณ์น้ำท่วมโดยเร่งด่วนต่อไป
โอกาสนี้ นายกรัฐมนตรี ได้ร่วมบริจาคเงินจำนวน  100,000 บาท สมทบเข้า “กองทุนเงินช่วยเหลือผู้ประสบสาธารณภัย สำนักนายกรัฐมนตรี” ด้วย พร้อมกล่าวถึงวัตถุประสงค์การจัด “งานประชารัฐร่วมใจ ใต้ร่มพระบารมี” ในครั้งนี้ว่า จากสถานการณ์น้ำท่วมครั้งใหญ่ในพื้นที่ภาคเหนือและภาคตะวันออกเฉียงเหนือ จากผลกระทบของพายุดีเปรสชั่นเซินกาในขณะนี้ ส่งผลให้ประชาชนภาคเหนือและอีสานได้รับความเดือดร้อนทั้งในด้านที่อยู่อาศัย การดำรงชีวิตและการประกอบอาชีพ รัฐบาลและทุกภาคส่วนไม่ได้นิ่งนอนใจต่อเหตุการณ์ที่เกิดขึ้น โดยพร้อมระดมความช่วยเหลือจากทุกภาคส่วน เพื่อช่วยแก้ไขปัญหาและบรรเทาความเดือดร้อนของพี่น้องประชาชนอย่างรวดเร็วที่สุด โดยวันนี้ รัฐบาลได้จัดงาน “ประชารัฐร่วมใจ ใต้ร่มพระบารมี” ขึ้น เพื่อขอเชิญชวนหน่วยงานต่างๆ ทั้งภาครัฐ ภาคเอกชน และประชาชน ร่วมกันบริจาคเงินเพื่อนำไปช่วยเหลือบรรเทาทุกข์ให้แก่พี่น้องประชาชนผู้ประสบภัยน้ำท่วมทั้งภาคเหนือและภาคอีสาน โดยทุกคนสามารถร่วมบริจาคเงินเข้า “กองทุนเงินช่วยเหลือผู้ประสบสาธารณภัยสำนักนายกรัฐมนตรี” บัญชีธนาคารกรุงไทย สาขาทำเนียบรัฐบาล เลขที่บัญชี 067-0-06895-0 โดยสามารถบริจาคผ่านทางรายการนี้ ซึ่งมีการเปิดหมายเลขโทรศัพท์พิเศษ 30 คู่สาย หรือจะบริจาคผ่านช่องทางปกติก็ได้เช่นเดียวกัน เพื่อให้ทุกคนได้ร่วมเป็นส่วนหนึ่งของการส่งต่อความปรารถนาดีไปยังพี่น้องประชาชนที่เดือดร้อนในพื้นที่ประสบภัยต่อไป
สำหรับเงินบริจาคดังกล่าว คณะกรรมการกองทุนเงินช่วยเหลือผู้ประสบสาธารณภัย สำนักนายกรัฐมนตรีจะประสานงานกับจังหวัดต่างๆ ให้เสนอขอรับการสนับสนุนแก่ผู้ที่ได้รับความเดือดร้อน ตามหลักเกณฑ์ต่างๆ ทั้งในเรื่องค่าวัสดุในการก่อสร้าง ซ่อมแซมบ้านเรือนที่เสียหาย ค่าเครื่องใช้อุปโภคบริโภคที่จำเป็นต่อครอบครัว รวมทั้งค่าจัดการศพในกรณีมีผู้เสียชีวิต เป็นต้น ทั้งนี้ ผู้บริจาคเงินเพื่อช่วยเหลือผู้ประสบภัยสามารถนำหลักฐานการบริจาคไปใช้แสดงการขอลดหย่อนภาษีได้ต่อไป
พร้อมทั้ง นายกรัฐมนตรี ได้กล่าวแสดงความหวังว่า พลังน้ำใจและความร่วมมือร่วมใจของประชาชนคนไทย ตลอดจนความช่วยเหลือของหน่วยงานทุกภาคส่วน จะเป็นกำลังใจสำคัญให้ประชาชนในพื้นที่ที่ประสบภัยสามารถผ่านพ้นวิกฤตินี้ไปได้ และกลับมาดำเนินชีวิตได้อย่างปกติสุขอีกครั้งหนึ่ง รวมถึงฝากกำลังใจไปยังเจ้าหน้าที่ที่ปฏิบัติงานดูแลช่วยเหลือประชาชนผู้ประสบอุทกภัยในพื้นที่ด้วย
สำหรับภายในงาน“งานประชารัฐร่วมใจ ใต้ร่มพระบารมี” ได้มีการเปิดหมายเลขโทรศัพท์พิเศษหมายเลข    0 2356 - 0303 ระบบอัตโนมัติ 30 คู่สาย เพื่อให้ประชาชนได้โทรศัพท์เข้ามาร่วมบริจาคในขณะดำเนินรายการ โดยมีพิธีกรชื่อดังร่วมดำเนินรายการ อาทิ นางสาวปนัดดา วงศ์ผู้ดี นางสาว เพ็ญพรรณ แหลมหลวง  นายโศภณ นวรัตนาพงษ์ และพิธีกรคู่ขวัญจากรายการคู่ข่าวเสาร์-อาทิตย์ ได้แก่ นางสาวรัชนีย์ สุทธิธรรม และนายกำภู ถูริภูวดล พร้อมทั้งเหล่าศิลปิน ดารา นักแสดง และศิลปินผู้พิการทางสายตา ร่วมขับร้องเพลงเพื่อเป็นขวัญกำลังให้กับเจ้าหน้าที่ผู้ปฏิบัติงานในพื้นที่ และผู้ประสบภัยน้ำท่วม ให้สามารถผ่านพ้นอุปสรรคปัญหาจากสถานการณ์น้ำท่วมไปได้ด้วยดี พร้อมร่วมรับโทรศัพท์ในการรับเงินบริจาคจากประชาชน หน่วยงานภาครัฐ ภาคเอกชนที่มีจิตศรัทธาร่วมบริจาคเงินในครั้งนี้ด้วย
นอกจากนี้ นายกรัฐมนตรี และคณะรัฐมนตรี ได้มาร่วมรับเงินบริจาคจากประชาชนและภาคส่วนต่าง ๆ ทั้งหน่วยงานภาครัฐ รัฐวิสาหกิจ ภาคเอกชน และสถาบันการศึกษา ที่เดินทางมามอบเงินบริจาคภายใน “งานประชารัฐร่วมใจ ใต้ร่มพระบารมี” ณ ทำเนียบรัฐบาล และร่วมรับโทรศัพท์รับเงินบริจาคจากประชาชนทั่วประเทศ  ซึ่งมีการถ่ายทอดสดผ่านทางสถานีโทรทัศน์ช่อง 9 สถานีวิทยุโทรทัศน์กองทัพบก ช่อง 5 สถานีวิทยุโทรทัศน์แห่งประเทศไทย กรมประชาสัมพันธ์ (NBT) และสถานีวิทยุกระจายเสียงแห่งประเทศไทย โดยจะนำเงินบริจาคดังกล่าวสมทบเข้า “กองทุนเงินช่วยเหลือผู้ประสบสาธารณภัย สำนักนายกรัฐมนตรี” เพื่อนำเงินไปช่วยเหลือและบรรเทาความเดือดร้อนของประชาชนผู้ประสบภัยน้ำท่วมในภาคเหนือและตะวันออกเฉียงเหนือโดยเร็วต่อไป
ทั้งนี้ ภายหลังการจัดงานดังกล่าว ประชาชนยังสามารถร่วมบริจาคเงินช่วยเหลือผู้ประสบอุทกภัยได้อย่างต่อเนื่องจนถึงเดือนกันยายน 2560 ผ่าน “กองทุนเงินช่วยเหลือผู้ประสบสาธารณภัย สำนักนายกรัฐมนตรี” ที่บัญชีธนาคารกรุงไทย จำกัด (มหาชน) สาขาทำเนียบรัฐบาล เลขที่บัญชี 067 - 0 - 06895 – 0  หรือติดต่อสำนักตรวจราชการ สำนักงานปลัดสำนักนายกรัฐมนตรี หมายเลขโทรศัพท์ 0 2283 4193  0 2283 4180 และ 0 2283 4931 หรือหมายเลขโทรสาร 0 2283 4193  0 2283 4180 และ 0 2283 4944 โดยผู้บริจาคเงินสามารถนำหลักฐานในการบริจาคไปใช้แสดงการขอลดหย่อนภาษีได้
