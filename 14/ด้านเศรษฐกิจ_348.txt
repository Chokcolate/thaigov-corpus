รัฐบาลไทย-ข่าวทำเนียบรัฐบาล-การประชุมคณะทำงานประชารัฐชุดส่งเสริม SME Start up และSocial Enterprise ภายใต้คณะกรรมการสานพลังประชารัฐ

วันศุกร์ที่ 5 พฤษภาคม 2560
การประชุมคณะทำงานประชารัฐชุดส่งเสริม SME Start up และSocial Enterprise ภายใต้คณะกรรมการสานพลังประชารัฐ
การประชุมคณะทำงานประชารัฐชุดส่งเสริม SME Start up และSocial Enterprise ภายใต้คณะกรรมการสานพลังประชารัฐ
วันนี้ (5 พ.ค. 60) คณะทำงานประชารัฐชุดส่งเสริม SME Start up และSocial Enterprise ได้มีการประชุมเพื่อสรุปผลการดำเนินงานปี 2559 และวางแผนงานปี 2560 ร่วมกับเครือข่ายพันธมิตรทั้งภาครัฐและเอกชน กว่า 67 หน่วยงาน โดยที่ผ่านมา มีการดำเนินกิจกรรมต่างๆ อาทิ ในกลุ่ม SME Traditional มีโครงการพี่ช่วยน้อง โครงการสินเชื่อประชารัฐวงเงิน10000 ล้านบาท กลุ่ม Start Up/Innovation Driven Enterprise มีโครงการ Start Up Runway และกลุ่ม Social Enterprise มีการจัดตั้งบริษัทเพื่อสนับสนุนวิสาหกิจเพื่อสังคม
นอกจากนี้ได้มีการวางกรอบการส่งเสริมและพัฒนา SME ปี 60 อย่างครบวงจรทั้งด้านการเงินการพัฒนาผลิตภัณฑ์การเพิ่มผลิตภาพและการตลาด โดยมีนายสมคิด จาตุศรีพิทักษ์ รองนายกรัฐมนตรีด้านเศรษฐกิจ นายอุตตม สาวนายน รัฐมนตรีว่าการกระทรวงอุตสาหกรรม ในฐานะหัวหน้าภาครัฐ นายเจน นำชัยศิริ ประธานสภาอุตสาหกรรมฯ ในฐานะหัวหน้าภาคเอกชนร่วมประชุม ณ ศูนย์การประชุมแห่งชาติสิริกิติ์
