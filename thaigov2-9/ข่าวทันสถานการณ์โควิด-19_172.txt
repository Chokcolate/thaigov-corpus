คุณหญิงกัลยา เชิญชวนครูทั่วประเทศเข้าร่วมอบรมหลักสูตร Coding for Teacher (C4T ) มุ่งพัฒนาครู เรียนรู้ได้ด้วยตนเองผ่านระบบออนไลน์ [กระทรวงศึกษาธิการ]
คุณหญิงกัลยา เชิญชวนครูทั่วประเทศเข้าร่วมอบรมหลักสูตร Coding for Teacher (C4T ) มุ่งพัฒนาครู เรียนรู้ได้ด้วยตนเองผ่านระบบออนไลน์
เมื่อวันที่ 21 เมษายน 2563 คุณหญิงกัลยา โสภณพนิช รัฐมนตรีช่วยว่าการกระทรวงศึกษาธิการ บันทึกเทปประชาสัมพันธ์เชิญชวนครูทั่วประเทศเข้าร่วมอบรม “หลักสูตรอบรมการจัดการเรียนรู้วิทยาการคำนวณสำหรับครู Coding for Teacher” (C4T ) ภายใต้นโยบาย “การขับเคลื่อนการจัดการเรียนรู้โค้ดดิ้งในโรงเรียน”
โดยหลักสูตรอบรมการจัดการเรียนรู้วิทยาการคำนวณสำหรับครู Coding for Teacher” (C4T ) ได้ผ่านการรับรองจากสำนักงานคณะกรรมการข้าราชการครูและบุคลากรทางการศึกษา (กคศ.) สามารถใช้ประกอบการเลื่อนวิทยฐานะได้ ทั้งนี้ครูที่สนใจสามารถสมัครสมาชิกและอบรมฟรีได้ที่ https://teacherpd.ipst.ac.th ตั้งแต่วันที่ 1 - 15 พฤษภาคม 2563 ระยะเวลาการอบรมระหว่างวันที่ 1 พฤษภาคม - 30 มิถุนายน 2563
รมช.ศึกษาธิการ กล่าวตอนหนึ่งว่า เชื่อว่าคุณครูและผู้ที่สนใจ เมื่อได้เรียนรู้แล้ว และได้เห็นตัวอย่างกิจกรรมต่างๆ ก็สามารถที่จะนำไปประยุกต์ใช้กับบริบทชีวิตประจำวันของทุกคนได้ ดังคำที่ว่า Coding for All, All for Coding / Coding ง่ายกว่าที่คิด พิชิตยุคดิจิทัล
