รัฐบาลไทย-ข่าวทำเนียบรัฐบาล-ตอบข้ออภิปราย Coding

วันพุธที่ 31 กรกฎาคม 2562
ตอบข้ออภิปราย Coding
รมช.ศึกษาธิการ ตอบข้ออภิปรายเกี่ยวกับการเรียน Coding
เมื่อวันที่ 26 กรกฎาคม 2562 ณ ศูนย์ประชุมใหญ่ทีโอที คุณหญิงกัลยา โสภณพนิช รัฐมนตรีช่วยว่าการกระทรวงศึกษาธิการ ได้เข้าร่วมการประชุมร่วมของรัฐสภาครั้งที่ 4 สมัยสามัญประจำปี ครั้งที่ 1 โดยได้ตอบข้ออภิปรายของนางสาวกุลธิดา รุ่งเรืองเกียรติ สมาชิกสภาผู้แทนราษฎร พรรคอนาคตใหม่ ในฐานะสมาชิกรัฐสภา เกี่ยวกับนโยบายของรัฐบาลด้านศึกษาเรื่องส่งเสริมการเรียนภาษาคอมพิวเตอร์ (Coding)
รมช.ศึกษาธิการ กล่าวว่า รัฐบาลได้ให้ความสำคัญกับการเรียนรู้ Coding โดยกำหนดไว้ในนโยบายด้านการศึกษาของรัฐบาลที่แถลงต่อรัฐสภา และกระทรวงศึกษาธิการก็มีนโยบายส่งเสริมและสนับสนุนการสื่อสารภาษาต่าง ๆ ของเด็กไทย ทั้งภาษาไทย ภาษาต่างประเทศ ตลอดจนภาษาคอมพิวเตอร์ หรือ Coding ซึ่งเป็นทักษะภาษาใหม่ที่จะใช้สื่อสารกับคอมพิวเตอร์และเทคโนโลยีในอนาคต ทำให้เข้าใจการทำงานของปัญญาประดิษฐ์ (Artificial Intelligence: AI) และหุ่นยนต์ สอดคล้องกับการเตรียมกำลังคนของประเทศให้มีทักษะเท่าทันโลกยุคดิจิทัล มีอาชีพ ตลอดจนมีคุณภาพชีวิตที่ดีขึ้น
ขณะนี้กระทรวงศึกษาธิการพร้อมที่จะขับเคลื่อน Coding ทั้งในเรื่องของอุปกรณ์ และบุคลากรที่มีความเชี่ยวชาญจากสำนักงานส่งเสริมการสอนวิทยาศาสตร์และเทคโนโลยี (สสวท.) ที่สามารถอบรมให้กับครูวิทยาศาสตร์และเทคโนโลยีที่สมัครใจและมีความพร้อมจะเรียนรู้เรื่อง Coding และแน่นอนว่า จะไม่เป็นการเพิ่มภาระของครู แต่หากดำเนินการอย่างต่อเนื่อง จะช่วยลดความเหลื่อมล้ำในการเรียนรู้ Coding ของเด็กในพื้นที่ห่างไกลกับเด็กในเมืองได้ด้วยเทคโนโลยีที่มีความทันสมัย
ทั้งนี้ ได้รับทราบว่าขณะนี้ญี่ปุ่นได้ประกาศให้ Coding เป็นนโยบายสำคัญของชาติ รวมทั้งกำหนดให้นักเรียนในระดับชั้นประถมศึกษาทั่วประเทศเรียนภาษา Coding ตั้งแต่ปี 2020 เป็นต้นไป เช่นเดียวกับจีนที่ประกาศให้การจัดการเรียนการสอน Coding เป็นนโยบายสำคัญของประเทศเช่นกัน
