วันอาทิตย์ที่ 9 สิงหาคม 2563
พิมพ์
“Thai SELECT” สัญลักษณ์นี้การันตีคุณภาพอาหารไทย
--
#ไทยคู่ฟ้า “Thai SELECT" เป็นตราสัญลักษณ์ที่ใช้ในการรับรองร้านอาหารไทย ซึ่งเป็นข้อบ่งชี้ว่าร้านอาหารนั้น ๆ สามารถปรุงอาหารไทยได้ตามมาตรฐาน มีวิธีการปรุงอย่างไทย ใช้วัตถุดิบและเครื่องปรุงของไทย เมื่อนักท่องเที่ยวเห็นตราสัญลักษณ์ จะช่วยเสริมความมั่นใจในการบริโภค นอกจากนี้ ร้านอาหารที่ผ่านการรับรอง จะได้รับการส่งเสริมและประชาสัมพันธ์จากกรมพัฒนาธุรกิจการค้า เช่น แอปพลิเคชัน Thailand Tourism Directory
.
ในปีนี้ กรมพัฒนาธุรกิจการค้า เปิดรับสมัครร้านอาหาร เพื่อเข้าคัดเลือกรับการรับรองตราสัญลักษณ์ Thai SELECT โดยมีร้านอาหารทั่วประเทศสนใจสมัคร 500 ร้าน และมีร้านที่ผ่านเกณฑ์มาตรฐาน 207 แห่ง ซึ่งต้องมีคุณสมบัติ ได้แก่ มีรายการอาหารไทยในร้านไม่น้อยกว่า 80% ปรุงอาหารตามมาตรฐานของไทย ใช้วัตถุดิบและเครื่องปรุงของไทย มีรสชาติ และคุณภาพอาหารยอดเยี่ยม ตกแต่งร้านสวยงาม และบริการเป็นเลิศ
.
สำหรับสัญลักษณ์ Thai SELECT มี 3 ประเภท คือ
1.Thai SELECT PREMIUM เป็น Thai SELECT SIGNATURE จะต้องได้รับ 90 คะแนนขึ้นไป
2.Thai SELECT เป็น Thai SELECT CLASSIC ได้รับคะแนนระหว่าง 75 - 89 คะแนน
3.Thai SELECT UNIQUE ได้รับคะแนนระหว่าง 75 - 89 คะแนน และต้องมีเมนูอาหารที่เป็นอัตลักษณ์ของท้องถิ่น
.
ปัจจุบันมีร้านอาหารได้รับตราสัญลักษณ์ไปแล้ว 987 ร้าน แบ่งเป็น Thai SELECT SIGNATURE จำนวน 10 ร้าน Thai SELECT CLASSIC 906 ร้าน และ Thai SELECT UNIQUE 71 ร้าน และจะเพิ่มขึ้นให้ครบ 2,000 ร้าน ครอบคลุมทั่วประเทศในอนาคต
.
ผู้ประกอบการที่สนใจ สอบถามรายละเอียดเพิ่มเติมได้ที่ Call center 1570 กรมพัฒนาธุรกิจการค้า
