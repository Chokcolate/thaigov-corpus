รัฐบาลไทย-ข่าวทำเนียบรัฐบาล-อนุทิน ย้ำขอผู้ประกอบการนำหลักฐานการเรียกเก็บเงินค่าหัวคิวโรงแรม

วันจันทร์ที่ 25 พฤษภาคม 2563
อนุทิน ย้ำขอผู้ประกอบการนำหลักฐานการเรียกเก็บเงินค่าหัวคิวโรงแรม
รองนายกรัฐมนตรีและรัฐมนตรีว่าการกระทรวงสาธารณสุข ขอให้ผู้ประกอบการนำหลักฐานการเก็บเงินค่าหัวคิวเพื่อเป็น State Quarantine และ Alternative State Quarantine ให้ตนที่กระทรวงสาธารณสุข หรือทราบเบาะแส แจ้งสายด่วนกรมสนับสนุนบริการสุขภาพ เพื่อดำเนินก
รองนายกรัฐมนตรีและรัฐมนตรีว่าการกระทรวงสาธารณสุข ขอให้ผู้ประกอบการนำหลักฐานการเก็บเงินค่าหัวคิวเพื่อเป็น State Quarantine และ Alternative State Quarantine ให้ตนที่กระทรวงสาธารณสุข หรือทราบเบาะแส แจ้งสายด่วนกรมสนับสนุนบริการสุขภาพ เพื่อดำเนินการตามกฎหมายโดยทันที
จากกรณีที่มีการแจ้งเบาะแสว่ามีบุคคลแอบอ้างเป็นเจ้าหน้าที่กระทรวงสาธารณสุข เรียกเก็บค่านายหน้าจากผู้ประกอบกิจการที่พักซึ่งรัฐจัดให้เป็นพื้นที่ควบคุมโรค (State Quarantine) หรือพื้นที่ควบคุมโรคแห่งรัฐทางเลือก (Alternative State Quarantine) นั้น ในวันนี้ (25 พฤษภาคม 2563) นายอนุทิน ชาญวีรกูล รองนากยกรัฐมนตรีและรัฐมนตรีว่าการกระทรวงสาธารณสุข ให้สัมภาษณ์ว่า ท่านนายกรัฐมนตรีจะไม่ยอมให้เกิดเรื่องแบบนี้ขึ้นได้ ขอให้ผู้ประกอบการนำหลักฐาน ภาพจากกล้องวงจรปิด หรือคลิปวีดีโอที่ถ่ายไว้มาให้ผมที่กระทรวงสาธารณสุข เพื่อจะได้ดำเนินการตามกฎหมาย
“เป็นไปไม่ได้ที่คนของกระทรวงสาธารณสุขจะทำเรื่องอย่างนี้ เพราะถูกสอนมาให้ดูแลคนไข้ ไม่ได้ถูกสอนมาให้โกง” นายอนุทินกล่าว
ทั้งนี้ หากพบเห็นบุคคลแอบอ้างในลักษณะดังกล่าว ขอให้นำหลักฐานหรือแจ้งที่สายด่วนกรมสนับสนุนบริการสุขภาพ หมายเลข 1426 หรือศูนย์รับเรื่องร้องเรียน กรมสนับสนุนบริการสุขภาพ หมายเลข 0 2193 7057 ในวันและเวลาราชการ เพื่อจะได้ดำเนินตามกฎหมายในกลุ่มผู้เกี่ยวข้องกับผู้กระทำผิด
