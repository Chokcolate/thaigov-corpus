วันศุกร์ที่ 29 พฤษภาคม 2563
สถาบัน APi จับมือแพลตฟอร์มออนไลน์-ห้างดัง เพิ่มช่องทางขายสินค้าเกษตรนวัตกรรม [กระทรวงพาณิชย์]
สถาบัน APi จับมือแพลตฟอร์มออนไลน์-ห้างดัง เพิ่มช่องทางขายสินค้าเกษตรนวัตกรรม
สถาบันส่งเสริมสินค้าเกษตรนวัตกรรม (APi) เดินหน้าช่วยเหลือผู้ประกอบการที่ผลิตสินค้าเกษตรนวัตกรรมที่ได้รับผลกระทบจากไวรัสโควิด-19 ให้มีช่องทางการจำหน่ายสินค้า ทั้งออนไลน์และออฟไลน์ เตรียมจับมือ 2 กรมหลักในพาณิชย์ ดันขายบนแพลตฟอร์ม JD Central , The Hub Thailand และ Thaitrade.com พร้อมร่วมมือกับสยามพารากอน และไอคอนสยาม เปิดพื้นที่ให้ขายทันที หากรัฐบาลคลายล็อก
นายกีรติ รัชโน อธิบดีกรมการค้าต่างประเทศ เปิดเผยว่า กรมฯ โดยสถาบันส่งเสริมสินค้าเกษตรนวัตกรรม (APi) ได้เดินหน้าช่วยเหลือผู้ประกอบการที่ผลิตสินค้าเกษตรนวัตกรรมที่ได้รับผลกระทบจากการแพร่ระบาดของไวรัสโควิด-19 ให้มีช่องทางการจำหน่ายสินค้า ตามนโยบายนายจุรินทร์ ลักษณวิศิษฎ์ รองนายกรัฐมนตรีและรัฐมนตรีว่าการกระทรวงพาณิชย์ โดยจะร่วมมือกับหน่วยงานภาครัฐ ห้างสรรพสินค้า และผู้ให้บริการด้านอีคอมเมิร์ซ เพิ่มช่องทางการจำหน่ายสินค้าทั้งทางออนไลน์และออฟไลน์
โดยในส่วนของช่องทางออนไลน์ สถาบัน APi ได้ประสาน 2 หน่วยงานภายในกระทรวงพาณิชย์ ได้แก่ กรมพัฒนาธุรกิจการค้า ในการผลักดันผู้ประกอบการสินค้าเกษตรนวัตกรรมเข้าร่วมจำหน่ายสินค้าบนแพลตฟอร์มออนไลน์ชื่อดัง 2 แห่งด้วยกัน ได้แก่ JD Central และ The Hub Thailand และกรมส่งเสริมการค้าระหว่างประเทศ เพื่อขยายโอกาสและเพิ่มช่องทางการค้าใหม่ๆ ผ่านเว็บไซต์ Thaitrade.com ซึ่งเป็นเว็บไซต์ตลาดกลางพาณิชย์อิเล็กทรอนิกส์ที่มีชื่อเสียงของไทย และคาดว่าจะช่วยเชื่อมโยงผู้ขายสินค้าไทยกับผู้ซื้อจากทั่วโลกได้ เพราะปัจจุบันถือเป็นช่องทางสำคัญในยุคที่คนต้องดำรงชีวิตในรูปแบบใหม่ (New Normal) และผู้บริโภคมีแนวโน้มที่จะเลือกซื้อสินค้าในแบบออนไลน์เพิ่มมากขึ้น
ส่วนตลาดออฟไลน์ สถาบัน APi ได้ร่วมกับศูนย์การค้าสยามพารากอน และไอคอนสยาม มีแผนนำผู้ประกอบการสินค้าเกษตรนวัตกรรมเข้าร่วมแสดงและจำหน่ายสินค้าในงาน “ตลาดนัดยิ้มสยาม” โดยจะจัดที่ รอยัล พารากอน ฮอลล์ ศูนย์การค้าสยามพารากอน และทรู ไอคอน ฮอลล์ ศูนย์การค้าไอคอนสยาม ซึ่งกำหนดวันจัดงาน จะจัดทันทีหลังจากที่ทางรัฐบาลคลายล็อก และให้สามารถจัดงานได้
“เป้าหมายของกรมฯ ในการเพิ่มช่องทางการจำหน่ายให้กับผู้ประกอบการสินค้าเกษตรนวัตกรรม เพื่อเป็นการช่วยเหลือผู้ประกอบการที่ได้รับผลกระทบจากไวรัสโควิด 19 ให้มีช่องทางการจำหน่ายสินค้าทั้งทางออนไลน์และออฟไลน์ และยังเป็นการกระตุ้นเศรษฐกิจภายในประเทศจากการจับจ่ายใช้สอย”นายกีรติกล่าว
